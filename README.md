React仿CNode博客网站练手demo

- 技术栈

1. 框架-react
2. 脚手架-create-react-app
3. 状态层管理-redux, react-redux
4. 路由管理-react-router-dom
5. 异步action-react-thunk
6. 网络请求-axios
7. UI框架-antd

- 实现功能

1. 首页分类查看博客列表，包括分页，数据刷新等功能
2. 查看博客详情，博客评论列表
3. 查看用户基本信息，发表和评论的帖子
4. 教程和关于视图

- API接口

CNode官网提供的接口：https://cnodejs.org/api
